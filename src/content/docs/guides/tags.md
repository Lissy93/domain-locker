### Links: Save Helpful Bookmarks

Links let you store domain-specific URLs directly in Domain Locker â€” so everything is in one place.

Some ideas:

- Admin dashboard  
- GitHub repo  
- Hosting provider  
- Analytics for the domain  
- DNS control panel  
- Search Console  
- CRM or support tool  

Each link has a title and a URL. You can add as many as you like per domain.  
View and manage them under [**/assets/links**](/assets/links), or on the domain detail page.

---

### Why Use Tags & Links?
With Domain Locker, you can categorise your domains under tags, and include links to all domain-related resources.

- Stay organised as your domain list grows  
- Find what you need quickly  
- Make it easier for your team to jump in  
- Reduce time spent digging for login pages or settings
